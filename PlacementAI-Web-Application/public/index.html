        About Project

        About This System

        🤖

        IBM watsonx.ai AutoAI Model

        The placement prediction model was trained using IBM watsonx.ai AutoAI — IBM's automated machine learning platform that automatically prepares data, selects algorithms, and optimizes model hyperparameters. The best-performing pipeline was deployed as an online deployment on IBM Cloud, making it accessible via a REST API endpoint.

        📊

        Rule-Based Guidance Engine

        Separate from the AutoAI model, a transparent rule-based guidance module analyzes each of the 10 input parameters against defined thresholds. This module provides human-interpretable feedback, identifies skill gaps, and generates actionable recommendations. It is completely independent from the machine learning prediction.

        🔒

        Secure Backend Architecture

        IBM Cloud credentials are stored in environment variables on the backend server (Node.js / Express) and are never sent to the browser. The backend uses IBM IAM token-based authentication with automatic token refresh, rate limiting, and security headers via Helmet.js.

        📚

        Technology Stack

        • ML Platform: IBM watsonx.ai AutoAI
        • Cloud: IBM Cloud — Watson Machine Learning
        • Backend: Node.js, Express.js
        • Frontend: HTML5, CSS3, Vanilla JavaScript
        • Security: Helmet.js, express-rate-limit, dotenv
